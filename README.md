# Mês a Mês

## Descrição

Mês a Mês é um WebApp simples criado para auxiliar no controle de gastos durante compras de supermercado. O usuário pode adicionar produtos, informar quantidade e preço unitário, e o sistema calcula automaticamente o valor total da compra.

O objetivo do projeto é oferecer uma ferramenta prática para acompanhar gastos enquanto se faz compras, além de servir como prática de desenvolvimento web utilizando tecnologias fundamentais do front-end.

## Funcionalidades

* Adicionar produtos à lista de compras
* Informar quantidade e preço unitário
* Cálculo automático do total por item
* Exibição do valor total da compra
* Remover itens individualmente
* Limpar toda a lista de compras
* Definir um orçamento mensal e visualizar o valor restante
* Armazenamento dos dados no navegador utilizando localStorage

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

```
mes-a-mes
```

## Como executar o projeto

1. Baixe ou clone o repositório.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` em um navegador.

Não é necessário instalar dependências ou utilizar servidor, pois o projeto funciona como um site estático.

## Objetivo acadêmico

Este projeto foi desenvolvido com foco em prática de conceitos fundamentais de desenvolvimento web, incluindo estruturação de páginas, estilização com CSS, manipulação do DOM com JavaScript e armazenamento de dados no navegador.
